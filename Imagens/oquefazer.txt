Atividade Prática 2 – HTML - Criação do Site “Germinaflix”
Antes de começar a atividade, crie uma pasta com o nome HTML2_8EGx, em que x é o número 
do grupo conforme definido pelo professor. Ex. Grupo 3 pasta = HTML2_8EG3
Cada grupo irá escolher 10 filmes / séries. Todos os filmes / séries deverão estar catalogados 
no site imdb.com (o grupo deverá ter o link da página de cada um dos filmes escolhidos para 
usar neste trabalho). 
Os grupos irão recortar a imagem disponível do filme /série para uso no site a ser desenvolvido 
pelo grupo. Usar a ferramenta de captura de imagens do próprio Windows
Os filmes serão divididos em 2 grupos:
Grupo 1 Mais Vistos
Grupo 2 Recomendados para você
*** ATENÇÃO ***
As páginas a serem criadas devem ter o título da aba do navegador personalizado indicando o 
seu conteúdo a um ícone.
Todas as imagens e ícones a serem usados devem estar numa pasta img abaixo da estrutura 
dos arquivos html

Parte 3: Entrega do trabalho
A pasta utilizada para este trabalho deverá ser entregue completa com todos os arquivos 
utilizados. Após encerrar o trabalho, o aluno deverá encontrar a pasta criada no explorador de 
arquivos clicar com o botão direito e escolher a opção Enviar, em seguida, escolher Pasta 
Compactada. Observar que um arquivo novo foi criado com o mesmo nome da pasta, mas 
como pasta compacta. ESTE ARQUIVO É O QUE DEVE SER POSTADO NO TEAM
